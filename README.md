# Personal Portfolio Website

## Overview

This is my personal portfolio website developed to showcase my profile, education, technical skills, projects, and contact details.

The project includes a modern responsive frontend design and a Spring Boot backend for handling contact form messages.

---

## Live Website

🌐 Portfolio Website:

https://vibodha-samarasinghe.github.io/PersonalPortfolio/

---

## Features

- Personal introduction section
- About Me section
- Education details
- Technical skills section
- Project showcase
- Download CV option
- Contact form
- Responsive design for mobile devices
- Frontend and backend integration
