# Personal Portfolio Website

## Overview

A personal portfolio website developed to showcase my profile, education, technical skills, projects, and contact details.

This project contains a responsive frontend website and a Spring Boot backend for handling contact form messages.

---

## Live Website

🌐 Portfolio Website:

https://vibodha-samarasinghe.github.io/PersonalPortfolio/

---

## Features

- Personal profile section
- About me section
- Education details
- Skills showcase
- Project showcase
- Download CV option
- Contact form
- Responsive design
- Spring Boot backend integration
